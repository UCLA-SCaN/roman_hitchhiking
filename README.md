# Roman HitchHiking

Roman HitchHiking is a methodology used to collect high-resolution, point-in-time
outage data for LEO Satellte Networks.

Roman HitchHiking is based off of [LEO HitchHiking](https://github.com/stanford-esrg/LEO_HitchHiking).

## Overview

### `data_collection`
Includes the scripts used to collect data using Roman-HitchHiking.

# Description of Datasets

## naive
Original Dataset
- Naive HitchHiking across all endpoints for 60 second (probing every second)

## naive_roman_rl
Original Dataset
- Methodology comparison
- 5 minute samples (probing every second)
- Methodologies:
  - Naive HitchHiking with n=4 (sampling 4 endpoints per every /24 subnet)
  - Roman HitchHiking with n=4
  - Roman HitchHiking with n=8

## roman-hh_may27
Original Dataset
- Roman HitchHiking with 8 source IPs
- 5 minute sample (probing every second)

## roman-hh_may28
Original Dataset
- Roman HitchHiking with 8 source IPs
- 5 minute sample (probing every second)

## roman-hh_may29
Original Dataset
- Roman HitchHiking with 8 source IPs
- 5 minute sample (probing every second)

## measurement_config_comp
Aggregated data from different measurement configurations:
- concurrent on a single IP
- concurrent on multiple IPs
- roman on a single IP
- roman on multiple IPs
across different sample sizes
